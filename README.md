# Finance Portfolio

A modern finance portfolio tracking application built with React. This application helps users manage and track their investments, expenses, and financial goals.

## Features

- Dashboard with financial overview
- Investment portfolio tracking
- Expense management
- Financial goals setting
- Data visualization with charts

## Project Structure

```
finance-portfolio/
├── src/
│   ├── components/
│   │   ├── Dashboard/

│   │   ├── Expenses/
│   │   └── common/
│   ├── hooks/
│   ├── services/
│   ├── utils/
│   └── App.js
├── public/
└── package.json
```

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/dltdot/finance-portfolio.git
   ```

2. Install dependencies:
   ```bash
   cd finance-portfolio
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Technologies Used

- React
- React Router
- Material-UI
- Chart.js
- Local Storage for data persistence
